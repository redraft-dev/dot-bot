## Notes

### First-time installation

1. Make sure you have yarn installed (`yarn -v` for testing). If you do not, use `npm i -g yarn` to install it.
2. Run `yarn` in the `frontend` folder to install dependencies.

### Building

1. Build the `backend` folder with `cargo build`.
2. Build the `frontend` folder with `yarn build`.
